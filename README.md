# Alphasky Monorepo

This repository contains Alphasky packages in a single monorepo.

## Included packages

- api
- assets
- data-synchronize
- dev-tool
- form-builder
- get-started
- git-commit-checker
- installer
- menu
- optimize
- page
- platform
- plugin-management
- revision
- seo-helper
- shortcode
- sitemap
- slug
- theme
- widget

## Notes

- Each package keeps its own `composer.json`.
- The `platform` directory contains the core modules used by `alphasky/platform`.
